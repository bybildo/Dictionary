# 📖 Dictionary (Language Learning Platform)

![Status: Work in Progress](https://img.shields.io/badge/status-work--in--progress-orange)

> **Примітка:** Цей проєкт знаходиться на стадії активної розробки. Функціонал постійно оновлюється та доповнюється. 
> 
> **Live Demo (Застаріла версія):** Ви можете переглянути попередню "заморожену" версію проєкту (тільки фронтенд) за посиланням: [bybildo.github.io/Dict2136/](https://bybildo.github.io/Dict2136/)

Full-stack веб-застосунок для вивчення іноземних слів за допомогою інтерактивних карток. Проєкт розробляється як інструмент для ефективного засвоєння нової лексики та планується як основа для дипломної роботи.

## Технологічний стек
* **Backend:** ASP.NET Core (Web API)
* **Frontend:** React + JavaScript/TypeScript
* **Database:** MS SQL Server + Entity Framework Core
* **Architecture:** N-layer Architecture (багатошарова архітектура)
